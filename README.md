# Prompt Enhancing Design

> A structured knowledge base of design patterns, methodologies, and best practices for crafting highly effective prompts for Large Language Models (LLMs).

---

## Overview

As LLMs become central to software products and knowledge workflows, the quality of prompts directly determines the quality of outputs. **Prompt Enhancing Design** is a reference repository that distills proven techniques for writing prompts that are clear, context-rich, and reliably effective across a wide range of models and use cases.

Whether you are building AI-powered applications, automating workflows, or exploring generative AI capabilities, this repository provides actionable frameworks you can apply immediately.

## Key Concepts

- **Prompt Structure** — How to organise a prompt for maximum clarity and control
- **Context Building** — Rules for injecting the right background information into a prompt
- **Instruction Precision** — Techniques for eliminating ambiguity from directives
- **Output Formatting** — How to steer the model toward the format you need
- **Iterative Refinement** — A disciplined loop for improving prompts over time
- **Role & Persona Patterns** — Using system-level framing to shape model behaviour

## Quick Start

1. Read [`design.md`](./design.md) for the complete framework.
2. Pick a **template pattern** that matches your use case (task prompt, conversational agent, chain-of-thought, etc.).
3. Apply the **context-building checklist** before writing your first draft.
4. Use the **evaluation rubric** in `design.md` to score and iterate on your prompt.
